Controls:
- Menu
	Two button(in green) indicate level of difficulty of the music track
	One button(in red) will close the game once clicked

- In-game
	Use microphone to control game input, speak or clap to begin a hit
	
	If the detection box is blue: not hit (no input from mic)
	If the detection box is light blue: a hit (has input but not a valid hit)
	If the detection box is yellow: a valid hit (has input when a beat arrived at the same time)

	One button(in green) will return the user to menu once clicked
